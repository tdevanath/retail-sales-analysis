# Retail Sales Analysis using Python

## Project Overview
This project performs descriptive and predictive analysis on a retail sales dataset to extract business insights and forecast future revenue trends.

## Features
- Descriptive analysis of sales performance
- Customer and category revenue analysis
- Regression model for sales prediction
- Time series forecasting
- Data visualization using matplotlib

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Structure
data/ – dataset  
src/ – analysis code  
requirements.txt – dependencies  

## How to Run
pip install -r requirements.txt  
python src/analysis.py
